# Кейс Бэтнетикс "Система симуляции и анализа забегов для тренера"
## Установка зависимостей 
### uv:
```shell
pip install uv
```
### Рабочие зависимости:
### Windows:
```shell
cd backend
uv venv .venv
.venv\Scripts\activate
uv pip install -e .
```
### Для разработки backend:
```shell
uv pip install -e ".[dev]"
```
